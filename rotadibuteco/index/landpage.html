1. Quem somos

O Rota di Buteco é um aplicativo criado para acompanhar visitas aos bares participantes do Comida di Buteco BH 2026. É um projeto independente, sem fins lucrativos, desenvolvido para uso entre grupos de amigos.

2. Dados que coletamos

Ao criar uma conta, coletamos:

  • Nome escolhido por você
  • Endereço de e-mail (usado apenas para login)
  • Avatar (emoji ou foto que você escolher e enviar)
  • Bares que você marcar como visitados
  • Notas e comentários que você registrar
  • Km informados por você

3. Como usamos seus dados

Seus dados são usados exclusivamente para gerar rankings, estatísticas e o feed de atividades visíveis aos outros participantes do app. Seu e-mail nunca é exibido para outros usuários.

4. O que é visível para outros usuários

  • Seu nome
  • Seu avatar ou foto de perfil
  • Bares visitados e notas dadas
  • Comentários que você escrever
  • Km rodados e estatísticas gerais
  • Emblemas conquistados

5. Onde seus dados ficam

Os dados são armazenados no Google Firebase (Firestore, Authentication e Storage), com servidores nos Estados Unidos. Fotos de perfil e visitas ficam armazenadas no Firebase Storage.

6. Seus direitos (LGPD)

  • Acessar seus dados a qualquer momento
  • Corrigir dados incorretos
  • Excluir sua conta e todos os seus dados incluindo fotos — disponível na aba Perfil
